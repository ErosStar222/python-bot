# Commands
* !help                   # Help menu
* !clear                  # Clear chat
* !sessions               # List open sessions
* !kill                   # Close sessions
* !servershell            # Run command on Server.py
* !shell < cmd >            # Run command
* !ddos < ip > < byte >       # Starts DDoS Attack
* !stopddos               # Stops DDoS Attack
* !miner < wallet >         # Start miner
* !stopminer              # Stop miner
